# Digital Name Card

Yoon Yohseob의 디지털 명함 웹페이지

## 기능
- 반응형 디자인 (모바일/데스크톱 호환)
- 모던한 UI/UX
- 소셜 미디어 링크 (LinkedIn, GitHub)
- 이메일 연결
- QR코드 스캔 최적화

## 배포
이 프로젝트는 Vercel에 배포하도록 설계되었습니다.

### Vercel 배포 방법
1. Vercel 계정 생성/로그인
2. GitHub 리포지토리 연결
3. 자동 배포 완료

## 파일 구조
- `index.html` - 메인 웹페이지
- `style.css` - 스타일시트
- `vercel.json` - Vercel 배포 설정

## 기술 스택
- HTML5
- CSS3 (Flexbox, Grid)
- Font Awesome Icons
- Google Fonts (Inter)
